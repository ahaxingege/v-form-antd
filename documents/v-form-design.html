# 介绍

v-form-design是一个表单可视化拖拽设计器，您可以使用该组件来设计表单，并且可以通过拖拽来改变表单的布局。

预览地址 (opens new window)

# 使用方式

<template>
  <v-form-design title="表单设计器"></v-form-design>
</template>

<script>
export default {
  name: 'formDesign'
}
</script>
